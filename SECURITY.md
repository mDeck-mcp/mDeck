# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in MCPDeck, please report it privately by emailing:

**codewithevilxd@gmail.com**

Please include:

- A description of the issue and steps to reproduce
- Affected version(s) and platform(s) (Windows, macOS, Linux)
- Any suggested fix or patch (optional)

We will acknowledge receipt and work on a fix as soon as we can.

## Scope

This policy covers the MCPDeck desktop application and its bundled `tsr` CLI binary. It does not cover third-party MCP servers listed in the Marketplace — those are maintained by their respective publishers.

## Supported versions

We only provide security fixes for the latest released version of MCPDeck. Please make sure you're running the latest release before reporting.
